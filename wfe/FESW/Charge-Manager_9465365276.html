Description

The charge manager state machine / code path is what controls charging of the battery.

When charge controller is connected to BMU we get a heartbeat

When we get the HV enable button we check IMD and interlock

If that is good to go to charge we close contactors and begin charging

We are also talking over CAN at this point

Charge to 90% and then switch to charge and balance until satisfied (check current level.  C/20 is probably good where C is capacity of pack).

Then we HV disable (open contactors)


We should also check cell temperature and make sure to slow down charging when the temp gets too high
