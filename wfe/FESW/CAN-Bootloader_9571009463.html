Project Description

Create a small command based bootloader that accepts commands over CAN. This will help in making remote programming easier.

Safety Features

These are some features that the bootloader must have.

  • Do not erase the bootloader
  • Check that the CRC of the downloaded application matches what was sent
  • Ensure there is a timeout before jumping to the application, giving enough time to send bootloader messages.
  • We should not download the wrong firmware to the wrong board.

Basic Commands to Support

  • Get Version – Get the current version of the bootloader
  • Read Memory Location – Read memory from the given address
  • Write Memory Location – Write memory to the given address
  • Go – Boot into the beginning of the application region.
