Brake Pressure Sensor Data Sheet:

ENG_DS_MSP300_A12.pdf

Part Number

M3031-000005-2K5PG

Scaling

  • Output Brake Pressure Sensor Voltage Range (after voltage divider): 1/3 - 3V
  • ADC Voltage Range: 0 - 3.3V converts to 0 - 4095 unsigned integer number
  • Unsigned integer range: 413 - 3723 (413.64 - 3722.73)
  • PSI Range: 1/3 - 3V converts to 0 - 2500PSI
  •  PSI = (2500/3309)Uint - (345000/1103)

Error Check: Ensure Uint does not exceed 3723 or drop below 413
