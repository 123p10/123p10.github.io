Project Description

Implement CAN communication with the motor controllers. This should enable the motor controllers, and allow sending torque demands.

Progress

Attempted to perform mc init procedure documented in ProCAN fundamentals document. Sent out all the ProCAN messages every 100 ms. Then, set the inverter command signal to 0x4 followed by 0x1 to enable bridge. Attempted to view on Gen5MC that the bridge was enabled, but it still showed disabled. The proCAN feedback inverter state seems to be always 0x15 (which according to the document is self test).
