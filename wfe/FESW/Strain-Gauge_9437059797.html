Strain Gauge    


Project Description

  • Research strain gauge operation and function
  • Create a catalog of strain gauges outlining the pros and cons of the various types
  • Design schematic of a strain gauge circuit
  • Build and test the schematic
  • Create a library for the component for STM boards

Supporting Information, Project Details

  • Basic reference for the operation of a strain gauge and the Wheatstone bridge: http://bit.ly/2yUmonY
  • A more advanced reference for a comparison of the different configurations of strain gauge circuits: http://bit.ly/1GTgsFo
  • Industrial reference site for strain gauges. Contains information about the different types of strain gauges used in industry.: http://bit.ly/2hkETIg
  • Detailed strain gauge rosette resource describing the main types of rosettes (tee, rectangular and delta). Has a "Rosette Selection Considerations" section :http://bit.ly/2h85NGA
  • Extremely detailed strain gauge catalog and documentation from HBM, a major metrology company: http://bit.ly/2gYmvEs
  • Another reference from HBM : http://bit.ly/2lKjCfC
