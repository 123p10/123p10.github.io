Project Description

We are to build the visual aspects of a graphical user interface (GUI) in order to display the real time results received from a telemetry system on a monitor run with Electron. Some of the results are to be displayed with numbers only, while others will occupy the screen as both numbers and graphs (if the option is selected) to evaluate a set of data received as a function of time.
