Project Description

Write a program and build a circuit that would enable the STM32F072RB to communicate with an external LED light or set of lights (strip).

 

Supporting Information, Project Details

  • This project will be based on this Arduino project

  • Hardware Requirements:

    • Breadboard

    • 3 x 10k Ohm resistors
    • LED strip
    • 3 x Logic Level N-channel MOSFETs
    • Jumper wires
    • 12V Power Supply

Requirements and Considerations

Desired Driver Functions:

  • Turn on
  • Turn off
  • Fade up/down

Basic Info: LED Driver

  • Regulate the power to an LED or string of LEDs
  • Provide a constant quantity of power to the LED as its electrical properties change with temperature 
  • Without a proper driver, the LED may become too hot and unstable, therefore causing poor performance or failure
  • May offer dimming via PWM and may have more than one channel for control of individual LEDs or colours

External LED Drivers

  • Currently in use in the TSAL 2018 Project
  • Can be directly controlled via voltage output
  • Makes it possible to control LEDs without software

Internal LED Driver

  • Our nucleo board has an option of 5V or 3V output
  • Requires additional circuit components (resistors and transistors) to increase voltage in order for it to power LED strips
