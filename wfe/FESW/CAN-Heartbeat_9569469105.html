Firmware : CAN Heartbeat

Created by Former user on May 12, 2020

The CAN heartbeat is another safety feature on the car. Each board periodically sends out a special message over CAN, called a heartbeat. As well, each board records the time it last received a heartbeat from every other board. If a board ever notices that the last time it received a heartbeat from a board is greater than the specified timeout, it sends out an error DTC. It is similar to a Watchdog Timer in this way, except it acts at the level of the car network instead of at the board level. 

Using the CAN heartbeat is automatic, but requires the watchdog task to be used. The watchdog task is the one that sends out the CAN heartbeat, and only does so if no watchdog errors have occured, in this way it links together the protection of the watchdog and the CAN heartbeat. As well, it is the watchdog task that monitors the heartbeats received from other boards.
