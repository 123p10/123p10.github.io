Firmware : Connecting to Eduroam

Created by Justin Vuong, last modified on Nov 11, 2021

Eduroam works on the IEEE 8021x standard.

To get the configuration to work you will need to create a connman configuration file:

Steps to connect

Open/create the file:

/var/lib/connman/BeagleBone.config

Add the following data and fill out your login credentials in the file:

[global]
Name = eduroam
Description = University of Waterloo Wifi
[service_peap]
Type = wifi
Name = eduroam
EAP = peap
Phase2 = MSCHAPV2
Identity = <Waterloo Email>
Passphrase = <Your Password>

When adding Identity and Passphrase make sure to exclude the <>

An example would look like this:

[global]
Name = eduroam
Description = University of Waterloo Wifi
[service_peap]
Type = wifi
Name = eduroam
EAP = peap
Phase2 = MSCHAPV2
Identity = j23vuong@uwaterloo.ca
Passphrase = myPass123

Double-check the id of the network:

sudo connmanctl
enable wifi
scan wifi
services

You will see an entry for eduroam that looks similar to the following:

connmanctl> services
*AO eduroam              wifi_fc45c312e5ed_656475726f616d_managed_ieee8021x


Now use the id to connect to the wifi. NOTE: this step might have to be run multiple times (around 4-5) and sometimes you will need to wait a couple of minutes before getting the connected prompt. The error “Connman Input/Output error” is common so keep trying to connect.

sudo connmanctl
connect wifi_fc45c312e5ed_656475726f616d_managed_ieee8021x

When you are connected you should see the following output:

connmanctl> connect wifi_fc45c312e5ed_656475726f616d_managed_ieee8021x
Connected wifi_fc45c312e5ed_656475726f616d_managed_ieee8021x

https://www.fis.gatech.edu/how-to-configure-bbw-wifi/
