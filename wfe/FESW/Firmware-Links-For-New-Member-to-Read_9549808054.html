Firmware : Firmware Links For New Member to Read

Created by Former user, last modified by Former user on Jan 29, 2020
  1. Confluence - Intro to Firmware - Quick Setup
    Follow these steps to set up your virtual environment Linux operating system.
  2. Formula Hybrid Competition Rules

    These are the guidelines that we need to follow when creating the car so it is important to know them.
  3. Git and Bitbucket tuorial https://www.atlassian.com/git/tutorials/what-is-version-control
    Git is how our team can have multiple programmers working on the same code without getting in each other's way. 
  4. CAN Bus tutorial https://www.kvaser.com/can-protocol-tutorial/
    Our car (and every other modern vehicle) uses CAN bus so that its various components can communicate with each other without an excessive amount of wiring.
  5. J1939 Protocol https://www.kvaser.com/about-can/higher-layer-protocols/j1939-introduction/
    Within CAN, we use the J1939 Protocol which you should understand.
  6. Unix/Linux tutorial

    We use Unix/Linux a lot in firmware and it is important to have a fundamental understanding.    
  7. C tutorial
    There is no link for this section because it is very similar to other common languages, for example C++. Try to find the differences between a language you are familiar with and start there. 
  8. Electrical System Form 2019 ESF
    Each year our team has to create a document outlining exactly how our car works. Although this is more on the electrical side, it is important that you know the different components within the car, what they do and how they work.
