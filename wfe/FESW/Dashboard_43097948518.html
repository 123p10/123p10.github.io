We are currently working on a dashboard in C++ that runs using the Qt framework. Documentation on how to compile Qt is found here: Compiling Qt

Python Dashboard [OLD]

The dashboard starts up using the files ~/.xinitrc ~/.bash_profile

To restart the dash kill the process running dashboard.py

ps -ef | grep dashboard
kill -9 #pid

If you want to run the dashboard from the cli run

export DISPLAY=:0
python3 /home/debian/firmware/beaglebone/app/wfe/dashboard.py
