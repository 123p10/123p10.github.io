Description

The LTC6804 or the updated LTC6811 is a battery stack monitor that can monitor (cell voltage and temperature) up to 12 cells in series for large battery packs.  They are also daisy-chain-able using a isolated SPI interface.  Using this we can monitor all 72 series groups of cells

See the AMS - Accumulator Management System page on more info about this IC
