Firmware : Safe Shutdown Procedure

Created by Owen Brake, last modified by Justin Vuong on Jul 31, 2022

Preliminary

Summary


Bring the car up to normal driving state then bring it back down to no power. Just ensure this is possible and has no issues.

Before starting the car:

  • Install segment maintenance disconnects (SMD).
  • Install HV disconnect.

Start the car:

  • LV On → HV Enable → EM Enable → EM Disable → HV Disable → LV Off
  • With the car at HV, make a measure of the tractive system measuring points using the Fluke1507 insulation resistance meter
    • HV+ to Gnd should be >2mega ohms, HV- to Gnd should be >2mega ohms

After turning off the car:

  • Remove HV disconnect.
  • Remove segment maintenance disconnect (SMD).




LV is turned on and off by the GLV switch

EDIT August 2021: This photo is outdated, the on state is parallel to the ground


HV and EM are turned on and off from the dashboard
