Firmware : Getting Started with Electron:

Created by Former user, last modified by Former user on Oct 13, 2017

Electron framework allows users to develop cross-platform desktop applications using JavaScript, HTML and CSS. The platform is based on Node.js and Chromium. To run electron on your computer it helps to have certain programs installed:

Using Windows:

  1. Install NPM and Node.js
    1. Download the windows installer from the Node.js website: Nodes.js® web site
      • This should install both Node.js and NPM
    2. Run the installer (the .msi file you installed dowloaded int he previous step)
    3. Follow the prompts of the installer (i.e accept user licence agreement, click the next button etc)
      1. Here is a photo of installer:
    4. Restart your computer - Node.js cannot run until you restart
    5. Test Node.js out by opening command prompt and type:
                node -v
      1. You should obtain the version of Node.js you downloaded which should be something akin to
        v0.10.35
    6. Test npm out by typing:
                npm -v  
      into terminal 
    7. You can continue testing this out by creating a test file to run it such as "hello.js" - instructions and more information can be found at:
      http://blog.teamtreehouse.com/install-node-js-npm-windows
  2. Install json
    1. In command prompt install the json framework using nom by typing in
           npm install -g json
      into the command prompt
  3. You can use any text editor of your choosing to write .js, .html and .json files which I believe can all be done on Notepad++
  4. Install electron using npm
    1. Open command prompt and type in the following code:
           npm install -g electron
      which should install the preinstalled electron binaries
  5. From here to run a simple electron program, i.e "Hello World" - follow the instructions as the structure of a electron program is outlined and it outlines the required files: https://electron.atom.io/docs/tutorial/quick-start/

 

Using Mac:

  1. Once again the first step is to install Node.js and npm (Note npm is installed when Node.js is installed)
    1. First you must install home-brew (if it is not yet install - check by typing
           brew doctor

      into the command line. You should see "Your system is ready to brew". Home-brew is a package manager and might be useful in the future though it is not completely necessary to install Node.js 

      1. Go into terminal and type
             xcode-select --install

        to install Xcode's tools (you can also install the full Xcode application though that is not necessary)

      2. From here install home-brew by typing
        ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

        into the command line. You may be prompted for a password but by default you can use the "brew" command to perform actions without needing sudo or your password

      3. You can check if home-brew is running correctly by typing in
        brew doctor

        into the command line. You should see "Your system is ready to brew". 

    2. Now that home-brew is installed on the Mac you should now install Node.js and npm using home-brew. To do this
      1. Open terminal and type
             brew update - this will update brew to contain the latest version of Node.js and npm
      2. Then type 
             brew install node - this should install node.js and npm for you
      3. Test is by typing 
             node -v - in your terminal the output should be a version number.
  2. Once again you should have a text editor that can create .html, .js and .json files. Textedit that is the default text editor on Mac is not very reliable in my opinion - you can try and install atom using the instructions on their site: https://atom.io
  3. Now you should install electron using npm - you can do this by
    1. Open terminal and type
      npm install -g electron and hit enter
  4. Once again to run a simple electron app you can follow the instructions on this website to get a basic understanding of how to develop apps using electron: https://electron.atom.io/docs/tutorial/quick-start/
