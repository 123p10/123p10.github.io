Firmware : Cell Tester

Created by Owen Brake, last modified on Jun 30, 2021

The Cell Tester Firmware is for testing the various battery cells on the car.

The Cell Tester software is broken up into 2 components:

  • Arduino (Measuring) Firmware
  • Python (Data Processing) Software

Python, Data Processing

Using the Python Executable

To run the data processing/reading, the user should enter the CellTester/Python directory.

Then the user should run python main.py <command> <arguments>

Test Command

This is the main command for testing the battery cell.

The test command takes in arguments detailed below that specify what datapoints to record and where to record them.

Arguments

  • -o <path_to_csv>
    • Provides the CSV file to write the Cell Data to, if the CSV file is non empty it will append cell data, preserving old cell data
    • If this is not provided by default it outputs to CellTester/Python/output/CellTester_<timestamp>.csv

  • -v

    • Tells the program to record the OCV( Open Circuit Voltage ) of the cells in the CSV
  • -r
    • Tells the program to record the internal resistance of the cells in the CSV
  • -t
    • Tells the program to record the temperatures of the cells
    • This includes the:
      • Ambient Temperature
      • Positive Terminal Temperature
      • Middle Terminal Temperature
      • Negative Terminal Temperature
  • -i
    • Tells the program to record the loaded Cell current measured

The test command will retrieve information from the Arduino and process that data into a human readable CSV for one cell.

Once the readings are taken the user is then prompted if they want to run the test again perhaps on a 2nd cell.

The user can input:

  • c
    • To "continue" and read data again from the Arduino potentially on the next cell
  • q
    • To "quit" and exit the program

The user can thus continuously test cells by attaching a cell, running the test, attaching a new cell after the test is complete and pressing 'c'.

Repeating this process over and over until all the cells of the battery are tested.


Requirements

  • Python 3
