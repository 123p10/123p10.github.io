Progress Outline and Documentation

Basic info:

  • Works on a basic correlation between metals and temperatures.
  •  As the temp increases, so does the metals resistance to the flow of electricity.
  • Most common specification is 100ohms, so at 0 degrees Celsius it would have 100ohms of resistance.
  • Most common metal used is platinum.
  • Generally, the more wires the better.

Circuit:

  • The voltage changes are too minimal to be used directly so an amplifier is needed.
  • Can use the Adafruit amplifier.
  •  Adafruit has a tutorial on the wiring and a library (pdf version here). 
  • The library will need to be ported.
  • schematic of amplifier.

RTD Comparisons by mounting types:


Free hanging:

  • Attached by a long wires with a probe-like sensor.
  • Can get into hard to reach places where you wouldn't put a circuit board. 
  • costs significantly more.
