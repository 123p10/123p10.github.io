Description

The temperature measurements from the AMS board need to be calibrated. To do this, it is necessary to read the data sheet for the thermistors, and determine the voltage bus bar boards will output as a function of temperature. 


The thermistors used are the Vishay NTCALUG02A103G. The data sheet is: 

A matlab script should be created to plot the temperature and voltage values based on the data sheet, and determine a high order polynomial (maybe 7th order) that fits the curve. The resistance values are in the data sheet, and the voltage can then be calculated by looking at the AMS schematic and determining the voltage source (see W20 AMS Respin).
