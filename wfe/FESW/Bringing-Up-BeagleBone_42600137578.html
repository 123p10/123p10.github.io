Firmware : Bringing Up BeagleBone

Created by Justin Vuong, last modified by User 48885 on Jan 26, 2022

This article will have all the steps required to flash and update the Buildroot system for the BeagleBone.

How to build an image

  1. Clone the BeagleBone Bitbucket Repository git clone git@bitbucket.org:waterloohybrid/2020_beaglebone_os.git

  2. Clone buildroot git clone https://github.com/buildroot/buildroot.git buildroot

  3. In the files, you should see the following files, “2020_beaglebone_os” and “buildroot”.

    In the buildroot file, go to configs and there should be a list of defconfig files.

  4. Locate the beaglebone_wfe_defconfig file by going through the 2020_beaglebone_os → configs.

  5. Move the beaglebone_wfe_defconfig file to buildroot → configs

  6. Open the terminal and enter the buildroot repository by running cd buildroot

  7. Run: ls config. This will show a list of defconfig files from the buildroot configs file. You will see that the beaglebone_wfe_defconfig file is on the list.

  8. Now run: make beaglebone_wfe_defconfig

  9. Run make. This command will start the build process and it may take a while for it to build.

  10. Once the build process has finished, run ls -la output/images to display the image files that have been built.

  11. Run lsblk to show a list of storage devices connected to your computer.

  12. To transfer the image to a different device. For instance, transferring to the beaglebone, run: sudo dd if=output/images/<insert image> of=/dev/<storage name> bs=1M.
    Note: bs (block size) is set to 1M, this is to show we’re copying the image to another device to be 1MB. The lower the image size, the faster it will take to transfer.

How to modify an image

Create a step-by-step guide:

  1. In the terminal, run crontab -e this opens the crontab file.

  2. Crontab allows you to schedule commands at a specific time. Below are the commands of scheduling crontab at any specific point in time.

    * * * * * command(s)
- - - - -
| | | | |
| | | | ----- Day of week (0 - 7) (Sunday=0 or 7)
| | | ------- Month (1 - 12)
| | --------- Day of month (1 - 31)
| ----------- Hour (0 - 23)
------------- Minute (0 - 59)

  3. Run crontab with BeagleBone defconfig file * * * * * /home/vagrant/buildroot
