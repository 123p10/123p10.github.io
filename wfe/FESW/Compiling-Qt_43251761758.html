Installations

First, you will have to make some installations on your machine.

sudo apt-get install build-essential
sudo apt-get install qtcreator
sudo apt install qt5-default
sudo apt-get install qt5-doc qtbase5-examples qtbase5-doc-html

Creating a Project with QMake

If you donâ€™t have a cpp file already created with a Qt project, you can make a file, say sample.cpp, and put the following in it:

#include <QApplication>
#include <QLabel>
#include <QWidget>

int main(int argc, char *argv[ ]) {
    QApplication app(argc, argv);
    QLabel hello("<center>Welcome to my first Qt program</center>");
    hello.setWindowTitle("My First Qt Program");
    hello.resize(400, 400);
    hello.show();
    return app.exec();
}

And then, in the directory, you can run:

qmake -project

This will create a .pro file.

Modifications to .pro File

As we want to compile out project from the command line, we will need to add the following line to our .pro file.

QT += gui widgets

Creating the Makefile

To create the Makefile, run

qmake sample.pro

where you replace sample.pro with the name of your .pro file that was created.

Running Make

Now that the Makefile is created, if you want to run the program, just run

make
