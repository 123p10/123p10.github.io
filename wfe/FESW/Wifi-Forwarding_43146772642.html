Firmware : Wifi Forwarding

Created by Justin Vuong, last modified by User c4eac on Sep 01, 2022

This is an UNTESTED way to forward WiFi from a host device to the BeagleBone, allowing for both the host device and the BeagleBone to remain connected to the internet while communicating wirelessly with each other.

Once again, this method is UNTESTED

  1. Enable packet forwarding on your device. For linux, this can be done by configuring the sysctl configuration file.

    1. Open the file with any text editor, the file is located at /etc/sysctl.conf

    2. Uncomment the following lines:

      1. net.ipv4.ip_forward=1

      2. net.ipv6.conf.all.forwarding=1

  2. Connect the laptop to the BeagleBone wirelessly

  3. Use sudo ifconfig to list all interfaces, make note of interface from the host device to the internet and the host device to the BeagleBone

  4. Run the following commands:

    1. iptables --table nat --append POSTROUTING --out-interface [YOUR HOST-INTERNET INTERFACE] -j MASQUERADE
iptables --append FORWARD --in-interface [YOUR HOST-BEAGLEBONE INTERFACE] -j ACCEPT

5. SSH into your BeagleBone terminal by running the command 

ssh root@192.168.7.2

6. Run the following command:

sudo route add default gw [IP ADDRESS OF YOUR HOST MACHINE OBTAINED FROM IFCONFIG PREVIOUSLY]
