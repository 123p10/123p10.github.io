Firmware : STM32CubeMX Installation - macOS

Created by Former user on Sep 29, 2018

Note

This is a slightly outdated guide.  As of writing, the stm32cubemx download contains files for windows, Mac and linux compatibility.  All that is needed is to scroll to the bottom of the page below and download stm32cubemx: https://my.st.com/content/my_st_com/en/products/development-tools/software-development-tools/stm32-software-development-tools/stm32-configurators-and-code-generators/stm32cubemx.license=1535579028450.html

Step-by-step guide

  1. If you know you have Java installed, proceed to step 2. If you are unsure, open the Terminal and type "java -version" and press Enter.
    1. If you get a message beginning with something resembling "java version "1.8"", proceed to step 2.
    2. If you get an error, go to this page and download Java under "Java SE Development Kit 8u144": http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html. Once Java is installed, proceed to step 2.
  2. Go to http://www.st.com/content/st_com/en/products/development-tools/software-development-tools/stm32-software-development-tools/stm32-configurators-and-code-generators/stm32cubemx.html
  3. Click "Get Software". This will bring you to the bottom of the page where the download button is.
  4. Click the "Get Software" button.
  5. Enter the required information in the pop up form and click "Download".
  6. One the .zip archive has finished downloading, extract it to your Desktop.
  7. Open Finder and locate the "en.stm32cubemx" folder on your desktop.
  8. Open the folder and right click on the "SetupSTM32CubeMX-4.22.1.exe" file (version number may be different).
  9. Hold down the option key and click on "Copy "SetupSTM32CubeMX-4.22.1.exe" as Pathname".
  10. Open the Terminal and type "java -jar " (with a space after jar) and then press Command+V to paste the path to the setup file.
  11. Press Enter
  12. After a few seconds, the STM32CubeMX should launch. Use it to install the application on your computer.
  13. Once you have it installed, you can open the application like any other by pressing Command + Space to bring up Spotlight Search and then typing "STM32CubeMX".

If you encounter any difficulties, don't hesitate to shoot me an email at mdcbonne@edu.uwaterloo.ca
