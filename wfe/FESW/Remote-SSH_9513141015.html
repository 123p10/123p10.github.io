Firmware : Remote SSH

Created by Former user, last modified on Sep 27, 2020

Introduction

I've set up access using a software virtualization tool called Zerotier. Basically, Zerotier facilitates peer-to-peer VPN connections, allowing devices on the Zerotier network to appear as though they are on the same LAN. This allows us to connect to the Beaglebone from anywhere relatively easily and without a whole lot of maintenence. More information on Zerotier can be found on their website and wikipedia. 

Installing

Follow the instructions at https://zerotier.com/download/.

Joining the server

Using the Zerotier GUI or CLI, join the address 8056c2e21cdc6742 (you may need sudo/admin privileges). The command through the CLI is as follows:

sudo zerotier-cli join 8056c2e21cdc6742


Zerotier should respond with a 200 Join OK if everything worked correctly. Before you can see other devices connected to the zerotier network (peers) you must be authenticated through the Zerotier network admin page. The credentials can be found at the bottom of this page. 

Connecting to the Beaglebone

I've assigned a static ip to the Beaglebone on the Zerotier network which can accessed with the following credentials:

Assigned IPUserPassword
172.24.7.2debiantemppwd

Now type ssh debian@172.24.7.2 and enter the password when prompted.

You should now be connected through SSH to the Beaglebone.

Zerotier Administration

Go to https://my.zerotier.com/. Log in through the Google account.

Authorizing a device: Go to the "Networks" tab and click on the "remote_ssh" network. Scrolling down should show a list of devices connected to the network. For each device, there is a checkbox under the Auth? column that authorizes devices to connect to other peers on the Zerotier network. Once that box is checked, that particular device is authorized to connect to the network. 

Troubleshooting

Connected to the Zerotier network but can't connect to any peers in the network: Disconnect from the school VPN server. Zerotier seems to have an issue working in parallel.
