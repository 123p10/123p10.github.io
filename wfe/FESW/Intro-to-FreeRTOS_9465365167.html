API Reference (Start with This)


https://www.freertos.org/static_menu.html#API_reference

FreeRTOS is well documented on their own project page. To start with, take a look at the API reference to get an understanding of the available functions in FreeRTOS. Most API functions have a code example associated with the documentation to help you understand how to use it. To start with, familiarize yourself with the following basic APIs and groups of APIs:

  • vTaskDelay

  • Queues

  • Semaphores/Mutexes

  • SoftwareTimers

  • Direct to Task notifications

FreeRTOS and Interrupts

Our processor family, the Cortex-M, has some specific configuration necessary for interrupts to work properly with Cortex-M. You probably won't need to do anything with this right away, but if you run into issues with interrupts and FreeRTOS its worth a read:

https://www.freertos.org/RTOS-Cortex-M3-M4.html
