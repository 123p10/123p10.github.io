Project Description

Create a graphical user interface (GUI) to facilitate the charging process. This will communicate with the Battery Management Unit (BMU) to receive data on the state of the HV battery, and control the charging process.

Motivation

Replace the Charge Cart Unit (CCU), which controls the charging of the car from four buttons on the board, with a graphical interface with runs on a team member's laptop and sends commands over the car and charger's CAN Bus. This would eliminate one board with microcontrollers from the boards needed each year, reducing the cost and effort of building the car charging system. 

Components

  • GUI display showing information about the battery and charging process status
  • Communication with BMU over CAN
    • Send control messages
    • Receive data messages
  • The GUI must interface with the charge cart relays- small relays to drive the larger charging contactors
  • The GUI must stop charging in a safe manner if the Charge Cart Emergency Stop button is pushed.
  • The GUI must interface with the purchased charger to begin and end charging.
