Firmware : Loading code onto Nucleo with Virtualbox (Windows and Ubuntu)

Created by Owen Brake, last modified on Sep 25, 2019

For Windows Users: Follow steps 4→12

For Ubuntu/Debian Users: Follow steps 1→12



  1. Install VirtualBox Extensions Pack
    1.  Goto https://www.virtualbox.org/wiki/Downloads
    2.  Download the VirtualBox X.X.XX Oracle VM VirtualBox Extension Pack
    3. In VirtualBox goto File->Preferences→Extensions
    4. Click add new packages at the right of the box
    5. Select and install the downloaded Extension Pack
  2. Run "sudo usermod -aG vboxusers username" in the terminal, replace username with your host username
  3. Log out and log back in
  4. Open VirtualBox
  5. Click Machines->Settings→USB
  6. Select USB 3.0 Controller
  7. Click the green plus icon on the right
  8. Select the STMicroelectronics STM32
  9. Click OK
  10. Run the VM
  11. If it worked there should be a new icon on the desktop representing the STM USB Drive
  12. Continue to follow the Blinking LED Tutorial
