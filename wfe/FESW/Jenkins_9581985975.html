Firmware : Jenkins

Created by zzakiull@uwaterloo.ca, last modified on Oct 04, 2020

Overview

Jenkins is an automation server which allows us to build, test, and deploy our software.

Currently, we're using it to run a nightly job that flashes the latest image. We're working to add jobs to test recent changes to repos in Bitbucket.

Our Jenkins server is currently found here: http://172.24.151.168:8080/

Master Server Setup

If we ever need to re-install Jenkins on an Ubuntu system, here are the steps to do so:

  1. Install Java

    sudo apt-get install openjdk-8-jdk

  2. Install a web server (e.g. Nginx)

    sudo apt-get install nginx

  3. Install Jenkins

    wget -q -O - https://pkg.jenkins.io/debian/jenkins-ci.org.key | sudo apt-key add -
sudo sh -c 'echo deb http://pkg.jenkins.io/debian-stable binary/ > /etc/apt/sources.list.d/jenkins.list'
sudo apt-get update
sudo apt-get install jenkins

    You may get a GPG error where the key can't be added. Refer to this page for more info, but in short, run the command below and then the other 4 commands above.

    sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys <the key>
  4. Type in the admin password

    By default Jenkins will run on port 8080. 

    You will be asked to enter the administrator password. You can find the password in the /var/lib/jenkins/secrets/initialAdminPassword file. You can use the cat command to display the password:

    cat /var/lib/jenkins/secrets/initialAdminPassword

After that, you will be asked to install some plugins (install the recommended ones). Next, you can set the admin username and password, and then you'll be taken to the Jenkins dashboard.
