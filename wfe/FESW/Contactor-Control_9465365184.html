Description

The Contactors are a set of 2 switches (think of them as high performance relays) that switch on/off power from the Accumulator to the rest of the system.

They are needed as per the rules (AIR (Accumulator Isolation Relays))

When powering-on the system we need to pre-charge the motors.

When powering-off the system we need to discharge the motors (they are essentially big capacitors)


Powering on:

Off → Precharging → Drive

Powering Off:

Drive → Discharge → Off
