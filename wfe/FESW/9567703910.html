Firmware : DCU (Dashboard Control Unit)

Created by Former user, last modified on May 07, 2020

The dashboard control unit (DCU) is responsible for the car dashboard, except the display which is management by the VCU. Therefore, the DCU performs a few main functions:

  1. Reading the buttons on the dashboard, the main ones being the EM toggle and HV toggle button 
  2. Controlling the LEDs on the dashboard, eg the EM enabled LED and HV enabled LED
  3. Sending out CAN messages related to button presses

State Machine (sortof)

Unlike most of the other boards, the DCU doesn't use the generic state machine driver. However, the functionality of the board is still documented using a state machine diagram, as it is a good way to document functionality. The state machine digram is shown on the following diagrams.net page, stored on the team google drive:
