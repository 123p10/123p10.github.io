Power Supply Setup

  1. Setup power supply so that it is in parallel mode on the 1st and 2nd ports
    1. Goto menu (shift + ?)
    2. Scroll down until you see Grp1, Grp2, ..
    3. Select Grp1
    4. Set mode to Parallel
  2. It should now say on the display Parallel in the second slot
  3. Set the voltage on the first terminal to be 12V, ~5A if you are spinning motors
    1. If just running boards without going up to HV or spinning motors something like 2-4A might be sufficient
    2. It's ok to set the current limit as ~5A each time though
  4. Set the power supply to meter mode so you can see how much current is being drawn and the voltage, to identify any issues.
  5. Turn the output on

Power On

HV Enable

1. Switch on the GLVMS (Grounded Low Voltage Master Switch). This means switching the red switch to the left.

2. Ensure that the red emergency stop buttons are not pressed in.

3. Press the button labelled "HV Enable", it should be the top button on the dashboard.

4. The HV Enable button should light up red

5. The TSAL should now be flashing red to indicate that the system is in High Voltage

EM Enable

1. Hold down the brake pedal (don't trip the BOTS(brake over travel switch)) quite a bit before starting.

2. Press the EM Enable Button this is the lower right button on the dashboard as shown.

3. The EM Enable button should light up blue

Startup Sanity Tests

Before doing anything on the car with EM Enable a few sanity tests should be performed.

This should hopefully prevent damage to the car due to incorrect setup or internal failure of components.

Electric Motor

Throttle

2. Very gently apply pressure to the throttle pedal(which is the pedal indicated in the below diagram).

3. With a very gentle pressure a high pitch whining sound should begin to be heard

4. Applying more pressure will result in rotation of the axle. This is the desired behavior and indicates that most systems are working as expected. When the motor is rotating correctly then we know the car has been able to pull High Voltage and communicate with the motors.
