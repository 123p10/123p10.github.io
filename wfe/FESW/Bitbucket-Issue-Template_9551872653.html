Firmware : Bitbucket Issue Template

Created by Former user, last modified on May 11, 2020

The following should be used as the template for all new Bitbucket issues. It is currently a work in progress, and new fields will likely be added.

Note that it is formatted for bitbucket markup, so even though it doesn't format on confluence it will once it is used on bitbucket


# Use this template for the issue
**Summary:** --Give a one sentence description of the issue--


**Subject Matter Experts:** --List names of people who are knowledgeable about this issue-- (use full names, not Bitbucket accounts in case account are changed/deactivated and also to make it easier to contact people on slack)


**Confluence Resources:** --Link to confluence page for this project, and other confluence pages that are useful for working on this issue--


**Relevant Rules:** --List any relevant FSAE rules that apply to this issue--


**Description:** --Give a description of the issue--


**Steps to reproduce:** --any steps required to reproduce this issue--


**Solution Strategy:** --if know at this stage, include a description of how the issue should be solved--
